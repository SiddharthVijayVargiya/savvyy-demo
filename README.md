# Savvyy – Smart Lingerie Demo

An interactive demo for **Savvyy**, India's lingerie brand.

## Features

- Smart AI chatbot with personalized recommendations
- Size calculator (band + cup)
- 50 products with images and horizontal scroll cards
- Occasion-based shopping flows
- Image upload — upload a product screenshot and the bot finds similar items
- Fabric education & upselling
- Order tracking simulation
- Hindi / Hinglish language toggle
- OpenAI GPT-4o-mini Vision API for image analysis

## How to run

Just open `savvyy.html` in any browser. No server needed.

## Image Upload Demo

Upload any product image from the `Savvyy...files/` folder.  
The chatbot will instantly identify the product and show similar items.
